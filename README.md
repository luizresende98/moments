# Moments
Projeto onde o usuário pode compartilhar momentos da sua vida. É possível cadastrar, editar e excluir os momentos, realizar comentários

## O que usei:
- HTML
- CSS
- Angular
- Typescript
- [API](https://github.com/matheusbattisti/curso_adonis_api_yt)
